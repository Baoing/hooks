# `useOnceClick`

`useOnceClick` 用于在指定的目标元素上监听一次点击事件。该 Hook 会在第一次点击时触发指定的回调函数。

## 参数

- (event: any) => void

## 返回值
无

__描述:__

传入一个函数，该函数用于注册点击事件处理程序。该处理程序将在指定的目标元素被点击时调用。

## 使用示例
```tsx
import React from 'react';
import {useOnceClick} from '@channelwill/hooks';
import {Button} from "@shopify/polaris";

const MyComponent = () => {
  const handleClick = useOnceClick(()=>{
    console.log("点击了一次")
  });

  return <div>
    <Button onClick={handleClick}>点击</Button>
  </div>;
};

export default MyComponent;
```
